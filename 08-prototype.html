Your PRD draft

This prototype ships with three pre-built critiques wired to the three sample PRDs — so you can demo it without an API key. To run it on real PRDs, paste the system prompt from 03-system-prompt.md into a Claude Project and use that as the real back-end.

Critique

Paste a PRD or load a sample, then click Critique this PRD.

Tip for interviews: open this page side-by-side with the rubric. Load the weak sample, run the critique, then load the strong sample and run it again. The point isn't that the tool works — it's that it calibrates differently between a weak and a strong draft. That's what you talk about.
