# PRD: Smart Reply in Inbox

**Author:** J. Doe, APM
**Status:** Draft v1
**Date:** Q3

## Overview

We want to add Smart Reply to the inbox to help users respond to emails faster. This will be an AI-powered feature that suggests quick replies based on the content of the email. It will be great for mobile users who are always on the go.

## Problem

Users are spending too much time replying to emails. Competitors like Google have Smart Reply and users love it. We need to keep up. Also, our DAU has been flat and this could help move it.

## Solution

Below each email, we'll show three suggested replies as buttons. The user can tap a button to send the reply, or edit it first. The suggestions will be generated by an AI model. We'll start with English and expand to other languages later.

## Success metrics

- Increase DAU
- Users send more emails
- Positive user feedback

## Risks

- AI could suggest something weird
- Users might not like it

## Rollout

We'll ship it to everyone in the next release.

## Timeline

- Design: 2 weeks
- Build: 4 weeks
- Ship: end of quarter
